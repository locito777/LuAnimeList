So benutzt du die Watchlist

  1. Öffne index.html in deinem Browser.
  2. Klicke auf einen Titel, um die Detailseite zu öffnen.
  3. Auf der Detailseite siehst du Beschreibung, Sprache und den Streaming-Link.

Einträge bearbeiten

Öffne die Datei data/anime.json in einem Editor. Beispiel:

{
  "id": "demon-slayer",
  "title": "Demon Slayer (Kimetsu no Yaiba)",
  "language": "Deutsch (Dub)",
  "watch_url": "https://dein-streaming-anbieter.de/titel",
  "description": "Kurze eigene Beschreibung …"
}

Speichere die Datei. Lade danach die Seite neu (F5).

Sprachen-Filter

Der Filter zeigt automatisch alle Sprachen an, die in anime.json vorkommen.

Neue Titel hinzufügen

  1. In anime.json einen neuen Block hinzufügen (achte auf doppelte id zu vermeiden).
  2. Die id sollte nur Kleinbuchstaben, Zahlen und Bindestriche enthalten.

Backup / Teilen

Du kannst den kompletten Ordner als ZIP packen und auf einen USB-Stick legen oder in der Cloud speichern.
